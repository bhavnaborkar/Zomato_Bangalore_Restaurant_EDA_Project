# Zomato_Bangalore_Restaurant_EDA_Project
Performed exploratory data analysis (EDA) on 12,000+ restaurant records to understand factors influencing ratings, cuisines, pricing, and location-based performance in Bengaluru. 
Cleaned and preprocessed large datasets by handling missing values, duplicates, and inconsistent categories using Pandas and NumPy. 
Created insightful visualizations using Matplotlib and Seaborn to identify trends in customer preferences across locations and cuisines.
